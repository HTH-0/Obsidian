Network IP : 네트워크 영역 식별 IP
Host IP : 호스트 간 식별 IP

SUBNET MASK
네트워크/호스트 IP를 지정하기 위해 사용되는 마스크
>필요한 네트워크 주소만 호스트 IP로 할당 할 수 있게 만들어 네트워크 낭비를 방지
>네트워크 영역을 늘리고 호스트 영역을 줄이겠다는 뜻

클래스풀
A : 255.0.0.0 
	> /8(약식)
	> 0.0.0.0 ~ 127.255.255.255 (주소범위)
B : 255.255.0.0
	> /16(약식)
	> 128.0.0.0 ~ 191.255.255.255
C : 255.255.255.0
	> /24(약식)
	> 192.0.0.0 ~ 223.255.255.255


	ex) 159.16.10.1
		: B클래스 범위, 159.16 네트워크자리, 10.1 호스트 자리
		: 서브넷마스크 255.255.0.0

통신 방식
1. Unicast : 일대일
	> 신뢰성 ↑, 회선 부담↑, 
2. Broadcast : 일대다
	> 보안↓, 회선 부담↓, 불필요한 트래픽 받을 수 있음
3. Multicast : 대상 선별 후 전달
