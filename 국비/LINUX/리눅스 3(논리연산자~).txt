논리연산자
	&&(and 연산)
		왼쪽 성공 > 오른쪽 실행, 왼쪽 실패 > 오른쪽 실행x
	||(or 연산)
		왼쪽 성공 > 오른쪽 실행x, 왼쪽 실패 > 오른쪽 실행

특수 문자

	* : 모든 문자, 길이 상관x
	? : 모든 문자, ?만큼의 길이

cat > a :
 컨트롤c : 작업 취소

리다이렉션 (ReDirection) :  명령의 출력을 변경 > 파일에 기록이 가능

	>, >>, <, <<  : 모니터에 표시되어야 할 출력물을 파일로 대체 
		> : 덮어쓰기, >> : 새로 추가함
		>/>> : 입력 방향 재지정, </<< : 출력 방향 재지정

error : 에러 출력

2> : 표준 에러



파이프라인 |
: 한명령어의 표준출력을 다른 명령어의 표준 입력으로 보내는 기능
