<피그마 팁>
Ctrl Alt c : 속성복사
레이어 겹쳤을 때 : Ctrl 누르고 우클릭
프레임 우클릭 > set as thumbnail : 썸네일 지정
오브젝트 Alt 누르고 이동 : 간격확인

프레임 Alt 드래그 : 프레임 복사

Ctrl 클릭 : 프레임 선택

shift a : 프레임 전환

shift r : 외곽에 안내선 표시
정확한 수치를 원할 때 : 하단 Rectangle을 이용해 길이를 조절
	> position : 설정

색 따오기
	> stroke > 스포이드

플러그인 사용
	> 우클릭 > PLUGIN 
	(추천)LUCI : 기호, UNSPLASH : 무료 이미지, Lorem ipsum : 문자열 생성(영문)/한글입숨(한글)
	skewdat : 도형왜곡	, arc : 텍스트를 둥글게, arrow auto : 플로우 생성

프레임 > layout grid
	주로 columns , 주로 모바일:4~6, 데스크탑:12
	margin : 좌우 여백, 	gutter : columns 사이사이 간격
