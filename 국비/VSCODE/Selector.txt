전체 선택자
	*{}

요소 선택자
	div{}
	p{}

id 선택자
	(앞에 #를 붙임)
	#id{}

class 선택자
	(앞에 . 을 붙임)
	.class{}

group 선택자
	(쉼표로 선택자들을 연결)
	#id , .class{}

### 자손선택자 때문에 선택자 생성 시 space 사용 x ###

자식 선택자 : >
	직접적인 자식 요소

자손 선택자 : 공백
	특정 요소의 모든 하위 요소


동위 선택자 : +, ~
	계층 구조에서 같은 부모 요소를 가지고 있는 요소
	+ : 하위 1개 선택자
	~ : 하위 모든 선택자

opacity : 투명도 설정


의사 선택자
	선택자에 추가하는 키워드

	btn:hover
	버튼위에 마우스커서를 둘 경우
	btn:active
	버튼을 누를 경우
	btn:visited
	버튼을 누르고 난 후


가상 선택자
	before : centent 영역 옆에 표시할 스타일
	after : centent 영역 뒤에 표시할 스타일
